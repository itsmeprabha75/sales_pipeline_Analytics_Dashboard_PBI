#**📈 Sales Pipeline Analytics Dashboard**

FP20 Analytics × ZoomCharts Mini Challenge

#**📊 Overview**

This project was developed as part of the FP20 Analytics × ZoomCharts Mini Challenge, focusing on Global B2B Sales Pipeline Analytics.

The dashboard provides a comprehensive view of the sales pipeline, sales team performance, and revenue segmentation across industries, regions, and product categories.

Using Power BI and ZoomCharts Drill Down visuals, the report allows business leaders and sales managers to monitor pipeline health, analyze sales rep activity, and identify revenue-driving segments.

The dashboard combines interactive exploration, clear storytelling, and business-focused insights to support data-driven decision making.

#**🧰 Tools and Technologies**

Power BI Desktop

ZoomCharts Custom Visuals (Drill Down PRO)

DAX (Data Analysis Expressions)

Microsoft Excel (Dataset Source)

Python (EDA & initial data exploration)

PowerPoint (Dashboard canvas background design)

#**📂 Dataset**

The dataset represents a global B2B sales pipeline for an enterprise SaaS and professional services company.

It contains information about sales deals, companies, sales activities, and pipeline stages across multiple industries and regions.

Key attributes include:

Deal ID, Deal Value (€), Deal Status, Win Probability

Sales Representative Details (Rep Name, Region, Role)

Company Information (Industry, Country, Company Size, Employee Count)

Pipeline Stages (Lead In → Qualified → Proposal → Negotiation → Contracting → Closing)

Activity Data (Calls, Emails, Meetings, Demos, Workshops)

Product Categories (Cloud Migration, Integration Services, Analytics & BI, etc.)

The dataset enables pipeline analysis, sales performance evaluation, and revenue segmentation insights.
